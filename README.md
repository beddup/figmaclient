
# FigmaClient

## What is FigmaClient

Figma Client is a simple plugin to 
- Fetch Figma design node data
- Download figma node image


## Installation

Install [OpenUPM CLI](https://openupm.com/docs/getting-started.html), then run:

```bash
openupm add com.beddup.figmaclient
```

or You can also download the source code directly and import it into your Unity project.

## Need a access figma token to fetch the data

[Manage personal access tokens](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)
